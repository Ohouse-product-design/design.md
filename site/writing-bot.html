01 · 무엇을 하나요?

5 modes

사용자 요청에 포함된 키워드를 보고 5가지 전용 작성 규칙 중 하나를 자동 적용합니다. 모든 답변에는 적용한 가이드와 이유가 함께 표시되고, 응답 전 톤앤매너·금지표현·글자수 체크리스트가 내부적으로 자동 검증돼요.

PUSH

앱푸시

제목 21자 / 본문 32자 이내, 본문은 (광고)로 시작. 자극적 표현·당신·오늘의집 직접 언급 금지.

앱푸시 · 푸시 · CRM
BANNER

배너 카피

추천 1 + 대안 7 = 8개 표로 제시 + 작성 의도. 쇼핑홈 IIF / 공통홈피드 / Binary 등 구좌별 글자수 가이드.

배너 · 홈피드 · IIF
UI

UI 문구

버튼·Empty·Snackbar·Tooltip·Error·Confirm. 메시지 유형별 어조(합쇼/해요/단호) 자동 적용. 최소 3가지 버전 제안.

버튼 · 모달 · 스낵바 · 툴팁 · UI · Empty
EXHIBITION

기획전

브랜드위크 / 브랜드캐스트 / Binary / 오하우스 에디션 별 구조와 글자수 가이드. 페르소나 3종(지나·세아·현정) 톤 선택 가능.

기획전 · 브랜드위크 · 브랜드캐스트 · Ohouse Edition
LAYER PDP

layer 상세

스펙이 아닌 사용 장면부터 시작. 한 문장 15~40자, 해요체. "최고·완벽한·궁극의" 같은 과장 표현 자동 차단.

layer · 레이어 · PDP · 상세페이지

02 · 사용 예시

Claude Code 프롬프트에 자연어로 요청하면 위 5가지 모드가 자동 선택됩니다. 키워드가 동시에 잡히면 layer > 앱푸시 > 기획전 > UI > 배너 순서로 우선 적용해요.

앱푸시 써줘. 봄맞이 패브릭 할인, 타깃 2030 여성
제목 21자 · 본문 32자 (광고) 시작 규칙으로 3개 이상 제안
  • 제목 "🌸 패브릭 봄맞이 시즌" / 본문 "(광고) {{nickname}}님, 봄 색감 가득한 신상 만나보세요"
  • 제목 "이번 봄, 거실이 달라져요" / 본문 "(광고) 패브릭 신상 최대 40% 할인 중이에요"
  • 제목 "패브릭 한 장으로 분위기 전환" / 본문 "(광고) 오늘 하루만 인기 패턴 모음전이에요"
"역대급/대박/실화" 같은 자극적 표현, "당신" 호칭, "오늘의집" 직접 언급은 자동 차단
배너 카피 써줘. 구좌 쇼핑홈 IIF, 브랜드 한샘
1줄 10자 내외 카피 8개를 표로 + 작성 의도 한 줄 설명
  • [추천] "오늘부터 한 끗 다른 거실"
  • "공간을 다시 쓰는 법"
  • "이 봄 가장 손이 가는 가구"
브랜드명·기획전명 직접 언급 금지, 마침표 금지, 느낌표 최대 1회 자동 검증
장바구니 Empty 안내 문구 써줘
안내 메시지(해요체) + 다음 행동 유도 + 3가지 톤 버전
  • "아직 담아둔 상품이 없어요. 마음에 드는 상품을 골라보세요" — 부드럽고 친근한 톤
  • "장바구니가 비어있어요. 인기 상품을 둘러볼까요?" — 액션 유도형
  • "장바구니에 담긴 상품이 없어요" — 간결 안내형
부정적 단어("아무것도 없습니다"), 이중부정 자동 회피
체크포인트 다듬어줘. '고밀도 구조 설계를 적용했습니다'
layer 톤 — 사용 장면 우선, 전문 용어를 경험으로 풀어 표현
  • 수정 후: "오래 사용해도 안정적인 구조예요"
  • 이유: 전문 용어("고밀도 구조 설계")를 사용 경험("오래 사용해도 안정적인")으로 변환. 합쇼체 → 해요체
"최고·완벽한·혁신적·압도적·궁극의·프리미엄" 자동 검출 → 대안 표현 제안

05 · 다운로드 + 설치

압축 파일을 받아 Claude Code 스킬 디렉토리에 풀면 바로 사용할 수 있어요. 별도 설치 명령이나 의존성은 없습니다.

writing-bot.zip

SKILL.md (~15KB) + evals/evals.json + evals/trigger-eval.json · 총 ~9KB

.zip 다운로드 Skill 문서 →
1

writing-bot.zip 다운로드

위 버튼으로 압축 파일을 받습니다. 다운로드 폴더에서 작업해도 되고, 바로 홈 디렉토리로 이동시켜도 돼요.

2

Claude Code 스킬 디렉토리에 압축 해제

개인 전역 스킬은 ~/.claude/skills/, 프로젝트 한정은 <프로젝트>/.claude/skills/ 아래에 두세요.

# 전역 (모든 프로젝트에서 사용) mkdir -p ~/.claude/skills unzip writing-bot.zip -d ~/.claude/skills/writing-bot # 프로젝트 한정 mkdir -p .claude/skills unzip writing-bot.zip -d .claude/skills/writing-bot

압축 해제 후 폴더 구조는 다음과 같아야 해요.

~/.claude/skills/writing-bot/ ├── SKILL.md └── evals/ ├── evals.json └── trigger-eval.json
3

Claude Code 재시작 + 스킬 인식 확인

Claude Code를 새 세션으로 시작하면 SKILL.md의 frontmatter에 정의된 description을 자동 인식합니다. 인식 여부는 다음으로 확인하세요.

# Claude Code 안에서 > /skills

목록에 writing-bot이 나오면 설치 완료. 키워드 매칭만으로 자동 호출되니 별도 명령은 필요 없어요.

4

자연어로 요청

아래처럼 한국어로 라이팅 요청을 하면 키워드가 자동 매칭되어 해당 모드가 적용됩니다.

> 앱푸시 써줘. 봄맞이 패브릭 할인, 타깃 2030 여성 > 배너 카피 써줘. 구좌 쇼핑홈 IIF, 브랜드 한샘 > 장바구니 Empty 안내 문구 써줘 > 브랜드위크 INTRO 써줘. 브랜드 덴비 > 체크포인트 다듬어줘. '고밀도 구조 설계를 적용했습니다'
